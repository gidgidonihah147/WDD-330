Ch4: Functions

Note that there are multiple ways to declare a function in Javascript. Pay close attention to the concept of callbacks as well.

  • Most common way to define a function is called "arrow"
  • All functions have a read only property called name
  • Adding functions through constructors are messy and best to avoid
  • Keep your code DRY - Do not repeat yourself. All parts of code should only be written once
  • You always need "()" to invoke the function. otherwise, you will just call the function, not run it
  • Any perameters will need to be within the "()" to call an argument with perameters
